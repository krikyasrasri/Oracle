The ASM instance in the Grid,was automatically starting on multiple server reboots,by adding the entry in Oratab.

+ASM:/ora_grid/product/v11.2.0/grid:N          # line added by Agent

Hence disabled it using srvctl command.
/ora_grid/product/v11.2.0/grid/bin
./srvctl stop ASM -f

$ srvctl disable asm
+ASM @ cds0stl32:/oracle/home

$ srvctl enable asm
